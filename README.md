Web demo
========

This project is a **very basic** web front-end demo.

Install
-------

```bash
npm install
```

Check syntax and rules
----------------------

```bash
npm run lint
```

Run in development mode
-----------------------

```bash
npm run start
```

Build for production
--------------------

```bash
npm run build
```

Run in production mode
----------------------

```bash
npm run serve
```
